Pour utilisé le site de gestion Prêt banque il faut :
1) Démarrer votre server local 
2) Importer le fichier nommé SQL qui se trouve dans le dossier Implémentation
3) Importer le fichier nommé donnée_test dans le même dossier
4) Dans le dossier site démmaré une des fichiers suivante :
    demandes.php
    formClient.php
    prets.php
    remboursement.php
